付録CDROM

以下に本CDROMのディレクトリ構成を示す。各ディレクトリには本書で紹介した
Javaのソースファイル（拡張子が.java），Javaのクラスファイル（拡張子が.
class），および行番号付きのソースファイル（拡張子が.txt）が収録されてい
る。

Source ディレクトリ中のファイルは，漢字コードが Shift_JIS で改行コードが 
CRLF である．

PDFファイル README.pdf には，以下のファイルのより詳細な情報や，ソースファ
イルのコンパイル方法についても記述されている．
